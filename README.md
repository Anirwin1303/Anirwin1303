# 💫 About Me:
I'm a recent Computer Science master's graduate from GWU, with a Mathematics-Computer Science background from UC San Diego. I'm drawn to problems combining mathematical reasoning and mathematical reasoning, and I like building things end to end, from systems to models. My work spans machine learning (LLM agent systems, models built from scratch to understand the math underneath) and lower-level engineering in C++ and Java, including a static-analysis tool I designed and shipped. I care about understanding why something works, not just that it does. I'm now looking for software engineering and DS/ML roles where I can build and keep learning.

## 🛠️ Projects

### REJD — Reverse Engineer Java Documentation Generator
`Java` · `Eclipse JDT` · `JavaFX` · `JavaScript/HTML` — *(Private repo — request access)*

- Built a static-analysis tool that parses Java source via Eclipse JDT's AST engine to auto-generate UML class and sequence diagrams at project, package, or class level — with binding resolution disabled, so it runs on incomplete codebases without bytecode or build tools.
- Shipped as both a standalone JavaFX desktop application and an Eclipse IDE plugin, the latter adding a collaborative annotation layer (threaded notes anchored to diagram elements, persisted as JSON) and an interactive JavaScript/HTML diagram canvas (programmatic SVG rendering, zoom, and drag).

### Multi-Agent QA System for Android Automation
`Python` · `Gemini API` · `Agent-S`

- Developed a full-stack, multi-agent QA system on the Agent-S framework with Gemini-powered Planner, Executor, Verifier, and Supervisor agents that autonomously execute and analyze mobile UI tests from natural-language goals in the `android_world` environment.

### Machine Learning Models from Scratch
`Python` — *(Private repo — request access)*

- Implemented five core ML models from scratch in Python without high-level libraries, matching Scikit-Learn baselines to demonstrate the underlying optimization and linear-algebra mechanics.

### Data Compression Tool
`C++` — *(Private repo — request access)*

- Engineered a lossless file-compression tool implementing Huffman encoding and bitwise I/O, building a Huffman tree from byte-frequency analysis to reduce file size by 70–80%.
- Used vectors and priority queues to process inputs up to 10 MB and compute per-byte frequencies efficiently.

### GPS Navigation Engine
`C++` — *(Private repo — request access)*

- Built a shortest-path navigation engine over a weighted undirected graph parsed from CSV input, using an adjacency-list representation to optimize time complexity.
- Implemented Breadth-First Search and Dijkstra's algorithm to compute shortest paths between weighted and unweighted nodes.
